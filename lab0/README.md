## Запуск проекта
1. Клонируйте репозиторий:
   ```
   git clone https://github.com/username/repository-name.git
   ```
2. Перейдите в папку с проектом
   ```
   cd repository-name/Lab0
   ```
3. Запустите программу:
   ```
     python src/task_with_number.py
   ```
